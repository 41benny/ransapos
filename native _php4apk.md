Tugas Anda:

1. Analisis struktur project saat ini:
   - Apakah menggunakan Blade, Livewire, atau Inertia?

2. Evaluasi kesiapan arsitektur:
   - Pisahkan logic bisnis dari layer view jika belum clean.
   - Identifikasi controller/service yang reusable untuk mobile.
   - Tandai bagian UI yang perlu redesign untuk mobile UX.

3. Buat laporan:
   - Risk assessment
   - Effort estimation (rendah / sedang / tinggi)
   - Dependency tambahan yang dibutuhkan untuk NativePHP
