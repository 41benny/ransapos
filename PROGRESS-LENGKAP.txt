================================================================================
TROUBLESHOOTING
================================================================================

Q: Transaksi gagal "Stok tidak mencukupi"?
A: Cek stocks table, pastikan quantity cukup
   > SELECT * FROM stocks WHERE product_id = X AND outlet_id = Y;

Q: Tidak bisa buka shift?
A: Cek apakah ada session open:
   > SELECT * FROM cash_sessions WHERE user_id = X AND status = 'open';
   Jika ada, tutup dulu atau update status = 'closed'

Q: Session number duplicate?
A: Generate logic sudah auto-increment, cek di:
   CashSessionService->generateSessionNumber()

Q: Selisih kas tidak muncul?
A: Pastikan JavaScript loaded, cek browser console
   Pastikan input id="actual_balance" ada

Q: Page not found?
A: Run: php artisan route:list
   Cek apakah route terdaftar

Q: View not found?
A: Cek nama file view sesuai dengan controller
   Cek folder structure: resources/views/...

Q: Server tidak jalan?
A: 1. Cek apakah ada process di port 8000:
      > netstat -ano | findstr "8000"
   2. Run server:
      > php artisan serve
   3. Atau pakai Laragon
