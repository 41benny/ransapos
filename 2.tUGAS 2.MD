## 🧩 KONTEKS SISTEM

Kita sedang membangun sistem **POS (Kasir)** dan **Back Office (Admin)** untuk retail/F&B (seperti kedai kopi atau resto kecil).  
Proyek ini sudah punya:
- 15 tabel utama (master, POS, inventory) ✅
- Semua migrations & models sudah selesai ✅
- Database SQLite sudah terbentuk ✅
- Belum ada routes, controller, atau tampilan (masih `welcome`)

Tugasmu sekarang adalah **membangun kerangka awal aplikasi (routing + layout + dashboard)** supaya aplikasi bisa diakses dan diuji di browser.

Gunakan **Bahasa Indonesia** untuk komentar dan dokumentasi.  
Gunakan **Laravel 12 + Blade + TailwindCSS** sesuai stack proyek ini.

---

## 🎯 TUJUAN TUGAS KEDUA

1. Ganti halaman utama (`/`) dari `welcome` menjadi redirect ke `/admin/dashboard`.
2. Siapkan dua area utama aplikasi:
   - **Back Office (Admin)** → prefix `/admin`
   - **POS (Kasir)** → prefix `/pos`
3. Buat layout dasar untuk masing-masing area:
   - `layouts/admin.blade.php`
   - `layouts/pos.blade.php`
4. Buat controller dan view untuk dashboard awal:
   - `Admin\DashboardController`
   - `POS\DashboardController`
5. Buat minimal tampilan daftar (list) sederhana:
   - Produk (`Admin\ProductController@index`)
   - Outlet (`Admin\OutletController@index`)
   - Supplier (`Admin\SupplierController@index`)

> Jangan ubah migration atau model dari tugas pertama.  
> Fokus hanya di route, controller, view, dan layout.

## ⚙️ STRUKTUR ROUTE YANG HARUS ADA

### 1. `routes/web.php`
- Ganti default `/` → redirect ke `/admin/dashboard`.
- Tambahkan group route:
  - `/admin` untuk Back Office (prefix `admin.`)
  - `/pos` untuk POS (prefix `pos.`)

Contoh:

```php
Route::get('/', function () {
    return redirect()->route('admin.dashboard');
});

Route::prefix('admin')->name('admin.')->group(function () {
    Route::get('/dashboard', [\App\Http\Controllers\Admin\DashboardController::class, 'index'])->name('dashboard');
    Route::resource('products', \App\Http\Controllers\Admin\ProductController::class)->only(['index']);
    Route::resource('outlets', \App\Http\Controllers\Admin\OutletController::class)->only(['index']);
    Route::resource('suppliers', \App\Http\Controllers\Admin\SupplierController::class)->only(['index']);
});

Route::prefix('pos')->name('pos.')->group(function () {
    Route::get('/dashboard', [\App\Http\Controllers\POS\DashboardController::class, 'index'])->name('dashboard');
    Route::get('/sales', [\App\Http\Controllers\POS\SaleController::class, 'create'])->name('sales.create');
});
