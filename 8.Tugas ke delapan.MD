## 0. KONTEKS

Sampai Tugas 7, sistem sudah punya:

- Modul POS (penjualan) + mutasi stok OUT.
- Modul Pembelian (purchase) + mutasi stok IN.

  - Belum ada modul **Kas & Bank**.
  - Belum ada pencatatan uang keluar/masuk.

**Tugas ini fokus:**
- Membuat modul **Kas & Bank** untuk admin.
- Menghubungkan pembayaran Purchase ke Kas/Bank.
- **Belum** bikin modul akuntansi penuh (jurnal, neraca, dll) → hanya buku kas sederhana.

## 1. TUJUAN TUGAS KEDELAPAN

1. Membuat tabel & modul **Akun Kas/Bank**:
   - Contoh akun: Kas Toko, Kas Besar, Bank BCA, Bank BRI.
2. Membuat tabel & modul **Transaksi Kas/Bank**:
   - Jenis: kas masuk, kas keluar.
   - Referensi opsional ke purchase.
3. Dari halaman Purchase:
   - Bisa “Catat Pembayaran” → membuat transaksi kas keluar + update `payment_status`.
4. Laporan sederhana:
   - Mutasi kas per akun + saldo akhir.

## 2. STRUKTUR DATABASE

### 2.1. Tabel cash_accounts

Buat migration & model:

- File migration: `create_cash_accounts_table.php`
- Tabel: `cash_accounts`

Kolom minimal:

- `id` (bigIncrements)
- `name` (string) → Nama akun: “Kas Toko”, “Bank BCA”
- `code` (string, unique) → KAS-TOKO, BANK-BCA
- `type` (enum: 'cash', 'bank')
- `is_active` (boolean, default true)
- `opening_balance` (decimal(15,2), default 0)
- `current_balance` (decimal(15,2), default 0)
- `notes` (text nullable)
- `created_by` (fk → users)
- timestamps

Model: `App\Models\CashAccount`
