## 🏗️ MODELS & BUSINESS LOGIC

### Customer Model
**File**: `app/Models/Customer.php`

**Relationships**:
```php
- hasMany(Sale) → sales
```

**Scopes**:
```php
- scopeActive($query) → is_active = true
- scopeByType($query, $type) → filter by customer_type
- scopeByTier($query, $tier) → filter by member_tier
- scopeVip($query) → customer_type = 'vip'
- scopeMembers($query) → customer_type IN ('member', 'vip')
```

**Key Methods**:

1. **addPoints($points)**: Menambah loyalty points & auto-upgrade tier
   ```php
   $customer->addPoints(100); // Add 100 points
   $customer->updateMemberTier(); // Auto check tier upgrade
   ```

2. **redeemPoints($points)**: Redeem loyalty points jika cukup saldo
   ```php
   if ($customer->redeemPoints(500)) {
       // Berhasil redeem 500 points
   }
   ```

3. **updateMemberTier()**: Auto-upgrade tier based on total spending
   - **Bronze**: < Rp 10,000,000 (default)
   - **Silver**: ≥ Rp 10,000,000
   - **Gold**: ≥ Rp 20,000,000
   - **Platinum**: ≥ Rp 50,000,000

4. **updateStats($amount)**: Update statistik setelah transaksi
   ```php
   $customer->updateStats($transaction_amount);
   // Updates: total_spending, total_transactions, last_visit
   ```

**Computed Attributes**:
```php
- type_badge → Badge color for customer type (gray/blue/purple)
- tier_badge → Badge color for member tier (orange/gray/yellow/purple)
- age → Calculate age from birth_date
- average_transaction → total_spending / total_transactions
```

**Helper Method**:
```php
private function generateCustomerCode(): string
{
    $lastCustomer = Customer::orderBy('id', 'desc')->first();
    $number = $lastCustomer ? $lastCustomer->id + 1 : 1;
    return 'CUST-' . str_pad($number, 6, '0', STR_PAD_LEFT);
}
```

## 📈 BUSINESS RULES

### Loyalty Program Logic

1. **Point Earning**:
   - Formula: 1 point per Rp 100 spending (1% cashback)
   - Example: Transaksi Rp 500,000 → 5,000 points
   - Auto-add points setelah transaksi selesai

2. **Point Redemption**:
   - Conversion: 1 point = Rp 10 discount
   - Minimum redeem: 100 points (Rp 1,000)
   - No expiry (configurable for future)

3. **Tier Upgrade (Automatic)**:
   - Based on **total_spending** (lifetime value)
   - Triggered by: addPoints() method
   - Upgrade rules:
     * Bronze → Silver: Rp 10,000,000
     * Silver → Gold: Rp 20,000,000
     * Gold → Platinum: Rp 50,000,000
   - **No downgrade**: Tier tidak turun meski spending stagnant

4. **Customer Types**:
   - **Regular**: Walk-in customer, no loyalty program
   - **Member**: Registered customer dengan loyalty points
   - **VIP**: Premium customer dengan special perks (future)
