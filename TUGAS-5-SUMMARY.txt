4. QUERY & AGREGASI
================================================================================

SALES REPORT:
-------------
Query untuk transaksi:
```php
Sale::with(['outlet', 'user', 'payments.paymentMethod'])
    ->where('status', 'completed')
    ->whereBetween('sale_date', [$dateFrom, $dateTo])
    ->where('outlet_id', $outletId)  // if filtered
    ->orderBy('sale_date', 'desc')
    ->get();
```

Agregasi cash vs non-cash:
```php
foreach ($sales as $sale) {
    foreach ($sale->payments as $payment) {
        if ($payment->payment_method_id == 1) {
            $total_cash += $payment->amount;
        } else {
            $total_non_cash += $payment->amount;
        }
    }
}
```

PRODUCTS REPORT:
----------------
Query dengan DB aggregate:
```php
DB::table('sale_items')
    ->join('sales', 'sale_items.sale_id', '=', 'sales.id')
    ->join('products', 'sale_items.product_id', '=', 'products.id')
    ->where('sales.status', 'completed')
    ->whereBetween('sales.sale_date', [$dateFrom, $dateTo])
    ->select(
        'products.name',
        'products.sku',
        DB::raw('SUM(sale_items.quantity) as total_qty'),
        DB::raw('SUM(sale_items.subtotal) as total_amount')
    )
    ->groupBy('products.id', 'products.name', 'products.sku')
    ->orderBy('total_amount', 'desc')
    ->get();
```

SHIFT REPORT:
-------------
Query shifts:
```php
CashSession::with(['outlet', 'user'])
    ->whereBetween('opened_at', [$dateFrom, $dateTo])
    ->where('outlet_id', $outletId)  // if filtered
    ->orderBy('opened_at', 'desc')
    ->get();
```

Payment breakdown per shift:
```php
DB::table('payments')
    ->join('payment_methods', 'payments.payment_method_id', '=', 'payment_methods.id')
    ->join('sales', 'payments.sale_id', '=', 'sales.id')
    ->where('sales.cash_session_id', $sessionId)
    ->select(
        'payment_methods.name',
        DB::raw('SUM(payments.amount) as total_amount'),
        DB::raw('COUNT(DISTINCT sales.id) as transaction_count')
    )
    ->groupBy('payment_methods.id', 'payment_methods.name')
    ->get();
```

Top products per shift:
```php
DB::table('sale_items')
    ->join('sales', 'sale_items.sale_id', '=', 'sales.id')
    ->where('sales.cash_session_id', $sessionId)
    ->select(
        'sale_items.product_name',
        'sale_items.product_sku',
        DB::raw('SUM(sale_items.quantity) as total_qty'),
        DB::raw('SUM(sale_items.subtotal) as total_amount')
    )
    ->groupBy('sale_items.product_id', 'sale_items.product_name', 'sale_items.product_sku')
    ->orderBy('total_amount', 'desc')
    ->limit(10)
    ->get();
```

E. VERIFY SQL MANUAL:

Check apakah angka di laporan sesuai dengan query manual:

1. Total penjualan hari ini:
   ```sql
   SELECT COUNT(*) as total, SUM(total_amount) as omzet
   FROM sales
   WHERE sale_date = '2025-11-07'
   AND status = 'completed';
   ```

2. Penjualan per produk:
   ```sql
   SELECT 
     p.name, 
     SUM(si.quantity) as qty,
     SUM(si.subtotal) as omzet
   FROM sale_items si
   JOIN sales s ON si.sale_id = s.id
   JOIN products p ON si.product_id = p.id
   WHERE s.status = 'completed'
   AND s.sale_date BETWEEN '2025-11-07' AND '2025-11-07'
   GROUP BY p.id, p.name
   ORDER BY omzet DESC;
   ```

3. Total shift:
   ```sql
   SELECT COUNT(*) as total_shifts, SUM(total_sales) as total_omzet
   FROM cash_sessions
   WHERE DATE(opened_at) = '2025-11-07';
   ```

15. QUERY COMPLEXITY
================================================================================

SIMPLE QUERIES (Good Performance):
-----------------------------------
✅ Sales report index:
   - Single table (sales)
   - 3 eager loads (outlet, user, payments.paymentMethod)
   - Simple where clauses
   - No complex join

MODERATE QUERIES:
-----------------
✅ Products report:
   - 3 table joins
   - Group by
   - Aggregate functions (SUM)
   - Order by calculated column

✅ Shift report index:
   - Single table (cash_sessions)
   - 2 eager loads
   - Date range filter
   - Collection calculations

COMPLEX QUERIES:
----------------
✅ Shift detail - Payment breakdown:
   - 3 table joins
   - Group by
   - Aggregate + count distinct
   - Subquery filter

✅ Shift detail - Top products:
   - 2 table joins
   - Group by
   - Aggregate functions
   - Order by + limit

All queries optimized dengan:
- Indexes pada foreign keys
- Where before join
- Group by di database (bukan PHP)
- Limit hasil jika perlu
