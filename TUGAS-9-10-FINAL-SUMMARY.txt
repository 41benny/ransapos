1. DATABASE - COMPLETE STRUCTURE
================================================================================

✅ Tabel Baru: coa_accounts
   CREATE TABLE coa_accounts (
       id BIGINT PRIMARY KEY AUTO_INCREMENT,
       code VARCHAR(50) UNIQUE NOT NULL,
       name VARCHAR(255) NOT NULL,
       type ENUM('income', 'expense', 'asset', 'liability', 'equity'),
       `group` VARCHAR(100) NOT NULL,
       is_active BOOLEAN DEFAULT TRUE,
       notes TEXT NULL,
       created_at TIMESTAMP,
       updated_at TIMESTAMP,
       INDEX(type),
       INDEX(is_active)
   );

✅ Update Tabel: cash_transactions
   ALTER TABLE cash_transactions 
   ADD COLUMN coa_account_id BIGINT NULL AFTER cash_account_id,
   ADD FOREIGN KEY (coa_account_id) REFERENCES coa_accounts(id) ON DELETE SET NULL,
   ADD INDEX(coa_account_id);

✅ Update Tabel: stock_mutations
   ALTER TABLE stock_mutations
   ADD COLUMN unit_cost DECIMAL(15,2) DEFAULT 0 AFTER quantity,
   ADD COLUMN total_cost DECIMAL(15,2) DEFAULT 0 AFTER unit_cost;

Purpose:
   - COA untuk klasifikasi pendapatan & biaya
   - cash_transactions linked ke COA untuk tracking biaya operasional
   - stock_mutations track HPP untuk perhitungan laba kotor

3. SERVICES
================================================================================

✅ StockService (UPDATED)
   app/Services/StockService.php
   
   Method Updated: reduceSaleStock()
   
   Logic:
      1. Ambil product.purchase_price sebagai unit_cost
      2. Hitung total_cost = unit_cost × quantity
      3. Simpan ke stock_mutations saat stok keluar
      4. HPP otomatis tercatat untuk setiap penjualan
   
   Contoh:
      Product: Espresso (purchase_price: 8.000, selling_price: 15.000)
      Sale: 2 cups
      → Revenue: 30.000
      → COGS: 16.000 (2 × 8.000)
      → Gross Profit: 14.000

✅ ProfitLossReportService (NEW)
   app/Services/ProfitLossReportService.php
   
   Method: generate(dateFrom, dateTo, outletId)
   
   Calculate:
      1. Total Revenue (dari sales)
      2. Total COGS/HPP (dari stock_mutations OUT)
      3. Gross Profit (Revenue - COGS)
      4. Operating Expenses by Group (dari cash_transactions dengan COA)
      5. Net Profit (Gross Profit - Total Expenses)
      6. Profit Margins (%)
   
   Return array dengan struktur lengkap untuk laporan
